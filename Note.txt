
Following the instruction to setup a local docker image for the mysql8.0 database:

https://medium.com/@crmcmullen/how-to-run-mysql-in-a-docker-container-on-macos-with-persistent-local-data-58b89aec496a

After that:

run the database:

docker run --restart always --name mysql8.0
-v /Users/andreagiassi/dev/mysql_data/8.0:/var/lib/mysql -p 3306:3306 -d -e MYSQL_ROOT_PASSWORD=root mysql:8.0

run the micro service app:

docker run --name micro2 -p 8090:8090 micro2
